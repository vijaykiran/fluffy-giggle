# fluffy-giggle

A Clojure client for [Vindinium](http://vindinium.org)

## Usage

- To run in arena:
`lein run -- -a {key}`

- To run in training mode try:
`lein run -- {key}`

## Options

| -a | run in arena mode |
| -t | run in training mode |
| -b | open the game in a browser |
